## Badge Styles (shields.io)

Shields.io supports several styles. Append one of these parameters to any badge URL:

| Style | URL Parameter | Preview |
|-------|---------------|---------|
| Default (flat) | _(none)_ | <img src="https://img.shields.io/badge/style-default-green" alt="default" /> |
| Flat square | `?style=flat-square` | <img src="https://img.shields.io/badge/style-flat--square-green?style=flat-square" alt="flat square" /> |
| Plastic | `?style=plastic` | <img src="https://img.shields.io/badge/style-plastic-green?style=plastic" alt="plastic" /> |
| For the badge | `?style=for-the-badge` | <img src="https://img.shields.io/badge/style-for--the--badge-green?style=for-the-badge" alt="for the badge" /> |
| Social | `?style=social` | <img src="https://img.shields.io/badge/style-social-green?style=social" alt="social" /> |

**Example: MIT with flat-square style**
```
https://img.shields.io/badge/License-MIT-yellow.svg?style=flat-square
```

## License Badge Maker

Replace variables between `{braces}`:

```
https://img.shields.io/badge/License-{NAME}-{color}.svg
```

**Common colors by license family:**

- **MIT / Permissive** → `yellow`, `green`, `brightgreen`
- **Apache / BSD** → `blue`, `orange`
- **GPL / Copyleft** → `blue`, `red`
- **Public domain** → `lightgrey`, `grey`
